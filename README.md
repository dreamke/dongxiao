# G调洞箫练习助手 V1.1

**by Dreamke**

---

## 简介

G调洞箫练习助手是一款专为洞箫学习者设计的网页应用，提供简谱显示、音准检测和气息训练三大核心功能，帮助您提升吹奏技巧。

---

## 功能介绍

### 1. 调音器

实时检测吹奏音高，帮助您校准音准。

#### 使用方法：
1. 点击"开始调音"按钮
2. 允许浏览器访问麦克风
3. 吹奏洞箫，观察屏幕显示的音符和音准偏差
4. 根据音分指示器调整气息：
   - **绿色** = 音准准确（±10音分内）
   - **黄色** = 音偏低
   - **红色** = 音偏高

#### 功能特点：
- **多调性支持**：支持G调、F调、D调切换
- **稳定计时器**：显示当前音保持的时长，帮助练习长音稳定性
- **频率波动阈值调节**：可选择20%/30%/40%/50%的容错范围
  - 阈值越低：计时器越容易重置，适合检测细微音高变化
  - 阈值越高：计时器越稳定，适合练习长音保持

#### 音符显示：
- **低音**：数字下方带圆点（如 5̣）
- **中音**：纯数字（如 5）
- **高音**：数字上方带圆点（如 5̇）

### 2. 节拍器

提供稳定的节拍参考，帮助您练习节奏感。

#### 使用方法：
1. 点击"开始"启动节拍器
2. 使用"+"/"-"按钮调节速度（BPM）
3. 选择拍号：2/4、3/4、4/4、6/8
4. 观察视觉节拍指示器，跟随节拍练习

#### 功能特点：
- **速度范围**：40-200 BPM
- **多种拍号**：支持常见节拍类型
- **视觉提示**：第一拍高亮显示
- **声音提示**：每拍都有清晰的提示音

---

### 3. 呼吸训练

科学的呼吸训练工具，帮助您提升气息控制能力。

#### 使用方法：
1. 点击"开始训练"启动呼吸计时器
2. 跟随屏幕提示进行呼吸：
   - **吸气**（蓝色）- 声音提示
   - **屏气**（黄色）- 声音提示
   - **呼气**（绿色）- 声音提示
3. 使用"+"/"-"按钮调节各阶段时长
4. 或选择预设方案快速开始

#### 预设方案：
- **基础 4-0-4**：吸气4秒，呼气4秒（无屏气）
- **方形呼吸 4-4-4**：吸气4秒，屏气4秒，呼气4秒
- **放松 4-7-8**：吸气4秒，屏气7秒，呼气8秒
- **长气息 5-5-10**：吸气5秒，屏气5秒，呼气10秒

#### 功能特点：
- **自定义时长**：可自由调节吸气/屏气/呼气时间
- **声音提示**：每个阶段切换时有不同音高的提示音
- **视觉反馈**：圆形动画和颜色变化直观显示当前阶段

## 使用建议

### 基础练习建议：
1. **音准练习**：使用调音器练习长音，目标是保持音准稳定3秒以上
2. **节奏练习**：使用节拍器从慢速开始（60 BPM），逐步提速
3. **气息练习**：每天进行3-5组呼吸训练，逐步延长呼气时间

### 进阶练习建议：
1. 将调音器和节拍器结合使用，练习音准的同时保持节奏稳定
2. 在呼吸训练中使用较高的阈值（40%-50%），专注于气息长度而非音准
3. 尝试在不同调性下练习，熟悉各调性的音高位置

## 技术说明

- **音频处理**：使用Web Audio API进行实时音频分析
- **音高检测**：采用自相关算法（Auto-correlation）检测基频
- **浏览器支持**：推荐使用Chrome、Edge、Firefox等现代浏览器
- **设备建议**：建议使用耳机以获得更好的音频反馈效果

---

## 注意事项

1. **麦克风权限**：首次使用调音器时需要允许浏览器访问麦克风
2. **环境噪音**：请在相对安静的环境中使用，避免背景噪音干扰
3. **浏览器缓存**：如遇功能异常，请尝试强制刷新页面（Ctrl+F5）
4. **移动设备**：支持手机和平板使用，但建议使用桌面浏览器获得最佳体验
